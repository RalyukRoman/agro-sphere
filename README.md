# AgroSphere
AgroSphere — це комплексна веб-платформа для автоматизації агровиробництва, що базується на технологіях Django та GeoDjango. 
Система дозволяє фермерам та агрономам планувати посівні кампанії, керувати складськими запасами та візуалізувати дані на інтерактивних картах.

## Основний функціонал
* Геоаналітика та ГІС: Візуалізація полів, складів та посівних площ на карті (PostGIS).
* Розумне планування: Оптимізація посівного плану за допомогою математичних алгоритмів для максимізації прибутку.
* Управління запасами: Журнал транзакцій для моніторингу товарно-матеріальних цінностей на складах.
